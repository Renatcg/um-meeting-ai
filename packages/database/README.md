# packages/database

Pacote de banco do UM Meeting AI.

## Responsabilidades

- Schema relacional inicial.
- Migrations.
- Extensao pgvector.
- Modelos para reunioes, participantes, mensagens, transcricoes, notas, embeddings e gravacoes.
- Seeds locais quando necessario.

## Banco planejado

- PostgreSQL
- pgvector para busca semantica em historico, notas e conhecimento comercial
