# Escopo do MVP

## Deve ter

- Criar uma reuniao.
- Entrar em uma sala LiveKit.
- Gerar token de participante pelo backend.
- Adicionar Jarvis como participante da sala.
- Transcrever a reuniao.
- Exibir transcricao ao vivo.
- Gerar notas e resumo final.
- Permitir que o Jarvis responda quando chamado.
- Enviar dicas privadas para um usuario marcado como comercial.

## Pode esperar

- CRM completo.
- Biblioteca avancada de playbooks.
- Analytics detalhado.
- Billing.
- Administracao multi-tenant completa.
- Edicao colaborativa de notas.
- Mobile app nativo.

## Principios do MVP

- Provar a experiencia de reuniao com IA participante.
- Manter o fluxo simples para demonstracoes.
- Persistir o minimo necessario para revisar reunioes.
- Separar desde cedo o canal publico das dicas privadas.
- Registrar eventos suficientes para auditoria e melhoria do agente.
